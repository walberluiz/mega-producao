# Mega Produção

Sistema web desenvolvido para substituir as planilhas de controle de produção utilizadas na Mega Sublimação.

## Objetivo

Criar uma ferramenta simples para:

- Cadastrar pedidos
- Planejar a produção da semana
- Gerar automaticamente a produção do dia
- Controlar pedidos concluídos
- Gerar relatórios para impressão

## Tecnologias

- HTML5
- CSS3
- JavaScript
- Git e GitHub
