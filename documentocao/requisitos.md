# Requisitos do Sistema

## Objetivo

Substituir as planilhas utilizadas atualmente pela empresa.

## Usuários

- Administrador
- Consultor

## Funcionalidades

### Cadastro de Pedido

Campos:

- Número do pedido
- Tipo
- Quantidade
- Data de entrega
- Reimpressão

### Planejamento Semanal

Mostrar os pedidos distribuídos por dia da produção.

### Produção do Dia

Gerar automaticamente a lista de pedidos que devem ser produzidos.

Permitir marcar pedidos como concluídos.

### Relatórios

Imprimir a produção do dia.

Mostrar pedidos atrasados.
